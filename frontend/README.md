## Getting Started

### Prerequisites

- Node.js 18+ 
- npm or yarn

### Installation

Install dependencies:
npm install

Start development server:
npm run dev

Build for production:
npm run build

Preview production build:
npm run preview

### Environment Variables

Create a .env.local file:
VITE_API_URL=http://localhost:8000/api/v1
